### Prerequisites

- Node.js (version 14 or later)
- npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/FabuTheFool/Color-Magic.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or if you're using yarn:
   ```sh
   yarn install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
   or with yarn:
   ```sh
   yarn dev
   ```
